CF Manifest Generator

About

I built this tool because I was tired of having to go to the documentation every time I needed to create a manifest file for my app. Just fill out the fields you want to populate, only app name is required, and click generate and you have the content for your manifest file.

Source

The source code for the tool is located on GitHub and is open sources under the Apache v2 license. Feel free to create pull requests for bug fixes or enhancements.
